# dbmigrate Copilot Instructions

## Project guidelines

- **CRITICAL: Write tests for EVERY change you make.** No feature, bug fix, or modification is complete without corresponding tests. This is non-negotiable.
- **MANDATORY: Run `make check` after EVERY change to ensure all tests pass 100% and there are NO dialyzer warnings/errors.** Never consider a change complete until you've verified all tests are green.
